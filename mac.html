Mac OS Installation.

First you need to install the Arduino IDE.

Next you will need Winchiphead signed drivers for CH340-CH341, which currently work for Mac OS X 10.9 to macOS 10.12.

Download the OLED I2C Arduino library and install it.

Finally, you need the latest Kidspeak software.
